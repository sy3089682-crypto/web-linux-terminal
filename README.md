# V-PLATFORM: Enterprise Cloud IDE

The world's most advanced web-based development platform. Built for speed, security, and global scale.

## 💎 Elite Features
- **Pro IDE Experience:** Full VS Code-style environment powered by the **Monaco Editor**.
- **AI Co-Pilot:** Integrated AI assistance for debugging, architectural advice, and code generation.
- **Extreme Persistence:** Zero-data-loss workspace with NVMe-speed Docker volume persistence.
- **Enterprise Dashboard:** Manage thousands of high-performance Linux instances with ease.
- **Google-Tier UX:** Fully responsive, animated, and resizable layout optimized for all devices.
- **Web Previews:** Instant public URLs for your hosted applications.

## 🚀 Getting Started (Enterprise Mode)

The entire platform is containerized for instant deployment:

```bash
docker-compose up --build
```

This will launch:
1. **Frontend (Vite/React):** The premium IDE cockpit.
2. **Backend (Node.js/Express):** The high-performance orchestration engine.
3. **Database (MongoDB):** Secure, scalable storage for users and instances.

## 🛠 Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS, Monaco Editor, Framer Motion, Lucide Icons, xterm.js.
- **Backend:** Node.js, Express, Dockerode, JWT, Bcrypt, Mongoose.
- **Infrastructure:** Docker, Docker Compose, Linux.
